# Emotion Detection Web App 😄

A CNN-based Emotion Detection system that detects human emotions from images using a web interface with camera capture and upload functionality.

---

## 🚀 Features

- 📸 Capture image using webcam
- 🖼️ Upload image from device
- 🤖 Emotion detection using CNN model
- 🎭 Displays emotion with emoji
- 💻 Clean and responsive UI

---

## 🧠 Tech Stack

- Python
- Flask
- TensorFlow / Keras
- OpenCV
- HTML, CSS, JavaScript
